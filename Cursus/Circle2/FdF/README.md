# FdF

## Usage:
>
	make
	./fdf <filename>

## Keys and actions:
* ```Arrow keys```: translate map.
* ```A``` and ```D```: rotate map.
* ```W``` and ```S```: tilt map.
* ```R``` and ```F```: increase/decrease elevation.
* ```,``` and ```.```: change views (net/dots/both).
* ```+```, ```-``` and ```mouse wheel```: zoom in/out.
* ```Q``` and ```ESC```: quit program.
